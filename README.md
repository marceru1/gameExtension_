# Jogo 2D – Projeto de Extensão (Godot)
Projeto desenvolvido como atividade de extensão acadêmica, utilizando a engine **Godot**, com foco no desenvolvimento de um jogo 2D.

Neste projeto, fui responsável pelo desenvolvimento do **personagem principal**, incluindo:
- Criação e implementação do personagem no jogo
- Configuração de movimentação e controles
- Integração do personagem com o cenário e a lógica do jogo
